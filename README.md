Esse código serve para criar um servidor HTTP com método POST e GET



## POST
<img width="1919" height="961" alt="image" src="https://github.com/user-attachments/assets/51892c62-78f5-4cfc-9256-6b47b754c93a" />
POST funciona como uma encomenda selada. Você não coloca as informações no "rótulo" (URL), mas sim dentro do pacote (o corpo da mensagem). Quando você clica em "Enviar" num formulário de cadastro, o navegador empacota seus dados e os entrega ao servidor de forma reservada.


## GET
<img width="1899" height="1015" alt="image" src="https://github.com/user-attachments/assets/1364dc1e-b319-4c5b-a5f1-c703beade14a" />
GET funciona como um cartão-postal. Quando você faz uma requisição GET, todas as informações que você quer passar para o servidor estão escritas "do lado de fora", na própria URL. Por ser uma forma direta de pedir dados, o navegador entende que essa é uma ação segura de ser repetida
